## 🚗 Sistema de Oficina Mecânica - Python

Sobre o Projeto

Este é um projeto pessoal desenvolvido com o objetivo de praticar e aprimorar meus conhecimentos na linguagem Python.

A proposta do sistema é simular o funcionamento básico de uma oficina mecânica, permitindo o cadastro de clientes, o gerenciamento da fila de espera dos veículos e a atribuição de serviços aos mecânicos disponíveis.

Atualmente o projeto possui funcionalidades básicas, mas continuará recebendo melhorias conforme eu avanço nos estudos e aprendo novos conceitos da linguagem.

---

## Origem da Ideia

A ideia deste projeto surgiu a partir da minha experiência no trabalho. Em alguns momentos, devido à correria do dia a dia, acabamos esquecendo de realizar ou registrar agendamentos de veículos, o que pode gerar atrasos e dificultar a organização dos serviços.

Durante as aulas de Análise e Desenvolvimento de Sistemas (ADS), comecei a pensar em uma possível solução para esse problema: e se existisse um sistema simples para organizar os agendamentos dos veículos e auxiliar no controle da oficina?

Foi então que decidi desenvolver este projeto como forma de praticar meus conhecimentos em Python enquanto buscava criar uma solução para uma situação que vivencio no ambiente de trabalho.

Além de servir como ferramenta de aprendizado, o projeto também representa uma oportunidade de transformar uma necessidade real em uma aplicação prática, que continuará evoluindo conforme adquiro novos conhecimentos e experiências na área de desenvolvimento de software.

---

## Funcionalidades da Versão 1.0 (02/06/2026)

- Cadastro de clientes e veículos.
- Registro do problema informado pelo cliente.
- Fila de espera para atendimento.
- Atribuição de veículos aos mecânicos.
- Visualização do status da oficina.
- Menu interativo via terminal.
